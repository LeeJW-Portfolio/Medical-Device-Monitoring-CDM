# Research-project
Research project during graduate school_Participating Researcher
==============================================================
## [국책과제] CDM 기반 의료기기 이상사례 능동적 모니터링 모델 개발 및 다기관 검증
> **과제번호: 21173미래기245**
> **※ 본 프로젝트는 국가 연구 과제 보안 지침에 따라 상세 코드 대신 분석 로직 및 데이터 정제 전략 요약본으로 대체합니다.**

## 📌 Project Overview
기존의 수동적인 부작용 보고 체계의 한계를 극복하기 위해, 병원 실사용 데이터(RWD)를 표준화된 공통데이터모델(CDM)로 변환하여 의료기기(담관 스텐트)의 이상사례를 자동 탐지하는 알고리즘을 설계하고 검증한 연구입니다.

## 📅 Period & Role
* **기간**: 2023.05 ~ 2023.12
* **역할**: 참여연구원 (코호트 설계 및 데이터 무결성 분석 담당)

## 🛠️ Tech Stack
* **Language**: SQL (CDM 기반 쿼리 로직 구축)
* **Tools**: GitHub (프로젝트 관리), 기술 문서 작성(Markdown)
* **Domain Knowledge**: CDM(Common Data Model), RWD, 의료기기 규제 과학

## 🚀 Key Activities
### 1. 코호트(Cohort) 설계 및 식별 로직 구축
* **Target 식별**: 소화기내과 전문의 자문을 바탕으로 담관 스텐트 삽입 환자 식별 로직 설계 (N=261)
* **Outcome 탐지**: 재폐색 의심군 탐지를 위한 지표(빌리루빈 수치 변화 등) 조작적 정의 수립

### 2. 다기관 교차 검증 및 알고리즘 최적화
* 설계한 SQL 쿼리를 다수의 의료기관 DB에 적용하여 알고리즘의 범용성 및 안정성 확인

## 💡 Key Insights: 데이터 무결성(Data Integrity) 규명
분석 과정에서 특정 기관의 결과가 '0건'으로 도출된 이슈를 단순 오류로 치부하지 않고 기술적으로 추적하였습니다.

* **문제 현상**: 특정 병원에서 담관 스텐트 재폐색 사례가 전혀 탐지되지 않음
* **원인 분석**: 알고리즘 자체의 결함이 아니라, 2021년 이전 EDI 데이터 매핑 과정의 시스템적 공백임을 규명
* **해결 전략**: 데이터의 시계열적 특성을 고려하여 2021년 이후 데이터에 가중치를 두는 정제 로직을 적용하여 분석의 신뢰도 확보

## 📈 Achievements & Professional Suggestions
* **기술적 유효성**: 다기관 검증을 통해 모델의 안정적인 사례 검출 능력 입증
* **정책적 제언**: 차세대 부작용 감시 체계를 위해 UDI(Unique Device Identification) 데이터 연동의 필수성을 논리적으로 제시
